# 🚀 GUIA PASSO A PASSO - TREINO OTIMIZADO

## 📋 PRÉ-REQUISITOS

### 1. Instalar Node.js
- Acesse: https://nodejs.org
- Baixe a versão **LTS** (recomendada)
- Execute o instalador
- Reinicie o computador se necessário

### 2. Verificar Instalação
Abra o **Prompt de Comando** (Win + R → cmd) e digite:
```cmd
node --version
npm --version
```
Deve aparecer algo como:
```
v18.17.0
9.6.7
```

---

## 🎯 PASSO A PASSO PARA INICIAR

### ✅ PASSO 1: Preparar o Ambiente
1. Navegue até a pasta do projeto:
   ```
   c:\Users\wemer\Documents\PROJETO 1 - INTELIGENCIA ART
   ```

2. **Clique duas vezes** no arquivo `setup.bat`

   Ou abra o Prompt de Comando na pasta e digite:
   ```cmd
   setup.bat
   ```

3. Aguarde a instalação das dependências (pode demorar alguns minutos)

### ✅ PASSO 2: Iniciar o Servidor
Após a instalação, **clique duas vezes** no arquivo `start.bat`

Ou no Prompt de Comando:
```cmd
start.bat
```

### ✅ PASSO 3: Verificar se Está Funcionando
1. Abra seu navegador
2. Acesse: **http://localhost:3000**
3. Deve aparecer a página inicial do TREINO OTIMIZADO

### ✅ PASSO 4: Testar o Sistema
1. **Página Inicial**: Preencha os dados pessoais
2. **Anamnese**: Responda sobre saúde e hábitos
3. **Análise Postural**: Faça upload das 4 imagens
4. **Admin**: Acesse http://localhost:3000/admin.html para ver cadastros

## 🔧 COMANDOS MANUAIS (se preferir)

Se quiser fazer manualmente:

```cmd
# Instalar dependências
npm install

# Iniciar servidor
npm start

# Ou para desenvolvimento (reinicia automaticamente)
npm run dev
```

---

## ⚠️ POSSÍVEIS PROBLEMAS

### ❌ "npm não é reconhecido"
- Node.js não foi instalado corretamente
- Reinicie o computador
- Reinstale o Node.js

### ❌ Porta 3000 ocupada
- Feche outros programas que usam a porta 3000
- Ou mude a porta no `server.js`

### ❌ Erro de permissão
- Execute o Prompt de Comando como **Administrador**

### ❌ Arquivos não encontrados
- Certifique-se de estar na pasta correta do projeto

---

## 📞 SUPORTE

Se tiver problemas:
1. Verifique se o Node.js está instalado
2. Execute `setup.bat` novamente
3. Verifique se a pasta `node_modules` foi criada
4. Abra o arquivo `database.db` para confirmar criação do banco
